# Java Swing Calculator

A desktop calculator built using Java Swing to practice GUI development and event-driven programming.

## Features

- Basic arithmetic operations (+, -, ×, ÷)
- Operator chaining (5 + 3 + 2)
- Decimal support
- Backspace button
- AC reset
- Clean GridLayout UI
- State-based calculation logic

## Concepts Practiced

- Java Swing (JFrame, JPanel, JButton, JLabel)
- GridLayout & BorderLayout
- ActionListener event handling
- State management
- Parsing strings to numbers
- DRY principle with button loop
- Calculator chaining logic
